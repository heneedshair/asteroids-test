# RENAR Reference

## Why a reference

The **standard** is for reading sequentially and for an assessor. The **reference** is for targeted lookup: one term, one field schema, one checklist before an audit, one ISO 29148 trace row.

You should not arrive here "from scratch." Start with [Core](../../core/en/renar-core.md) and [guide/00-quickstart](../../guide/en/00-quickstart.md); come back here when you need a table, a schema, or a printable kit.
