# RENAR Core — концептуальный обзор

## С чего начать

Если вы впервые сталкиваетесь с RENAR и хотите быстро понять о чём стандарт — **начните здесь**. Core — короткий обзорный документ для человека-читателя: что такое RENAR, зачем нужен, как работает на верхнем уровне, кому пригодится.

В Core **нет** технических подробностей: frontmatter артефактов, lifecycle-переходов, нормативных правил, схем валидации. Эти детали — в полном [RENAR Standard](../standard/README.md) (15 глав).

| Документ | Время | Содержание |
|---|---|---|
| [renar-core.md](renar-core.md) | ≤ 10 мин | Концептуальный обзор: что/зачем/как/кому/где читать дальше |

После Core, в зависимости от роли:

- **Если вы человек** — далее [guide/00 Быстрый старт](../guide/00-quickstart.md) для practical onboarding или маршруты по ролям из [renar-core.md](renar-core.md#маршруты-по-ролям).
- **Если вы AI-агент** — переходите напрямую к [Standard](../standard/README.md). Core рассчитан на человеческое чтение и для AI избыточен.

---

## Когда Core достаточен

Core — обзор, не conformance mode. RENAR conformance оценивается **только** относительно полного Standard (manifest §13.4, mandatory clauses §13.3). Core не вводит альтернативной нормативной модели.
